# Add Keyboard Controls

##### 7. In `game.js` setup variables for the different keys used to move characters.

```javascript
// File: game.js
// Copy
const keys = {
   w: keyCodes.W,
   s: keyCodes.S,
   a: keyCodes.A,
   d: keyCodes.D,
   up: keyCodes.UP,
   down: keyCodes.DOWN,
   left: keyCodes.LEFT,
   right: keyCodes.RIGHT,
   tab: keyCodes.TAB,
};
// End Copy
const keyCodes = Phaser.Input.Keyboard.KeyCodes;/*[*/
```
