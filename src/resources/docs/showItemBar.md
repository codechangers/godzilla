---
title: showItemBar()
tags: [customize]
---
This method temporarily shows the item bar.
## Parameters
**None**
## Returns
**None**
## Usage
You can use this method when you have created an item bar, you can use it if you want to be able to toggle the item bar on and off.
## Examples
### 1. 
```
//Link code/client/src/game.js
click() {
	g.showItemBar();
}
```
