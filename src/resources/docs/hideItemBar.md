---
title: hideItemBar()
tags: [customize]
---
This method temporarily hides the item bar.
## Parameters
**None**
## Returns
**None**
## Usage
You can use this method when you have created an item bar to be hid.
## Examples
### 1.
```
//Link code/client/src/game.js
click() {
	g.hideItemBar();
}
```
